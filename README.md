# clojure-scrapers

Clojure web scrapers for deployment to Heroku.

## Usage

Install [Leiningen](http://github.com/technomancy/leiningen).
Clone the repository.

	$ cd clojure-scrapers
	$ lein deps
	$ lein run -m clojure-scrapers.core
